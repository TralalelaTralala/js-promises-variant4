
Weather Dashboard — це веб-застосунок для отримання погодних даних з декількох API одночасно.

Проєкт демонструє роботу з:

- Promise
- Promise.race()
- setTimeout()
- catch()
- timeout
- асинхронними операціями
- обробкою помилок

У проєкті реалізована симуляція трьох погодних API з випадковою затримкою відповіді.
